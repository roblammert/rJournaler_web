# rJournaler_Web v1.0.1 Release Notes

Release date: 2026-03-08
Tag: v1.0.1

## Summary

rJournaler_Web 1.0.1 is a stabilization and deployment-hardening patch release for Windows-to-Windows Docker-host deployments, with weather refresh reliability fixes and improved installation/deployment guidance.

## Highlights

- Added dedicated installation guide:
  - `docs/INSTALLATION.md`
- Improved remote/packaged Docker deployment flow:
  - package + checksum + build + deploy scripts
  - stricter Docker command failure handling
  - local-image enforcement for image-based compose
- Weather refresh reliability improvements:
  - immediate fetch path when cache is missing
  - lock timeout and recovery improvements
  - Linux-compatible interpreter resolution in API weather fetch path

## Operational Fixes Included

- Compose run now avoids accidental registry pulls for local-tag images.
- Build scripts ensure images are loaded into local Docker store.
- Deploy script validates local image presence before compose up.
- PHP app image now includes required runtime components for weather subprocess execution.

## Recommended Post-Deploy Verification

- Confirm `app` and `worker` containers are running with tag `1.0.1`.
- Validate login + dashboard load.
- Validate weather endpoint while authenticated:
  - `/api/weather-session.php` returns either weather payload or explicit actionable error (not indefinite in-progress loop).
- Confirm migrations run cleanly from image-based deploy path.
