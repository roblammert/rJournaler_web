# rJournaler_Web v1.0.6 Release Notes

Tag: v1.0.6
Date: 2026-03-18

## Highlights
- Removed TOTP and trusted device authentication features for simplified login and maintenance.
- App version code is now auto-generated from the version string (e.g., W010006 for 1.0.6).
- Bug fixes and codebase cleanup for authentication and entry UID logic.

## Upgrade Steps
- Update all containers and configs to tag `1.0.6`.
- Confirm `app` and `worker` containers are running with tag `1.0.6`.
- Review `CHANGELOG.md` and `docs/INSTALLATION.md` for deployment details.

## Bug Fixes
- Removed all TOTP and trusted device code and database columns.
- EntryUid now builds version code from the version string automatically.
- All interface pills and deployment instructions now reflect 1.0.6.
- Docker compose and config files default to tag 1.0.6.
